# Union Debt Info Widget

This project is a widget for Scriptable iOS app. It fetches and displays information about the total amount owed by a user to Union Finance.

## Getting Started

These instructions will get you a copy of the project up and running on your local machine.

### Prerequisites

You will need to have the [Scriptable](https://apps.apple.com/us/app/scriptable/id1405459188) app installed on your iOS device.

### Installing

1. Copy the contents of `union_debt_info.js` file.

2. Open Scriptable app.

3. Tap on the plus icon to create a new script.

4. Paste the copied contents into the new script.

5. Save the script with a name of your choice.

6. Add a Scriptable widget to your iOS home screen and select the saved script.

### Usage

The widget will fetch the latest total amount owed, the due today amount, and the status (Overdue/Borrowing/Not borrowing) of the user to Union Finance every time it's refreshed.
